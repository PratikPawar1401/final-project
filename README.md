# Simple Interest Calculator

## Project Overview

This project is a Simple Interest Calculator implemented using a Bash script. It allows users to compute simple interest by providing the principal amount, rate of interest, and time period as inputs.

---

## Objective

The objective of this project is to demonstrate basic shell scripting concepts and user interaction through terminal-based input and output.

---

## Features

* Accepts user input for:

  * Principal amount
  * Rate of interest
  * Time period
* Performs calculation using the standard simple interest formula
* Displays the computed result in the terminal

---

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

---

## Technologies Used

* Bash Scripting
* Linux/Unix Shell

---

## How to Run

1. Open a terminal.
2. Navigate to the directory containing the script.
3. Make the script executable:

   ```bash
   chmod +x simple-interest.sh
   ```
4. Execute the script:

   ```bash
   ./simple-interest.sh
   ```

---

## Example Execution

```
Enter Principal:
1000
Enter Rate:
5
Enter Time:
2

Simple Interest = 100
```
